Curso de Stranger Things DevArt - Gustavo Campelo Projeto desenvolvido atráves de quatro aulas pelo DevArt #2 Nele aprendi os conceitos de animação com GSAP e ScrollTrigger, além de uma estrutura profissional de projeto com HTML e CSS. Além disso, o Gustavo Campelo mostrou como integrar essas tecnologias em um projeto real e como poderemos implementar essas técnicas em nossos próprios projetos
